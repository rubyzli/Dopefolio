Bizzlink - digital business cards

Full Stack Web Application, enabling users to seamlessly exchange virtual business cards through the convenience of QR code scanning.

Code Link

Project Overview

We built a Full Stack Web Application that does a lot. On the frontend, we used React.js, Bootstrap, and Material UI to make a user-friendly interface. People can make accounts, log in, and then create their own digital business cards by adding their info. These virtual cards can be shared with others by scanning a QR code â€“ just like those codes you see on products.

On the back part, we used Spring Boot, which helps connect everything. We also added Spring Security to keep data safe and Spring Data JPA to store user info in a PostgreSQL database. This way, people's data is kept secure, and they can easily manage and access their info. The whole setup lets our web app work smoothly, making it simple for folks to create, share, and manage their digital business cards hassle-free.
